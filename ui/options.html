Token da API Helena

Necessário apenas para enviar o link do cadastro pelo chat via Helena. O token fica só neste navegador (chrome.storage.local), não é enviado ao repositório.

Segurança: quem usa este perfil do Chrome pode inspecionar a extensão ou o armazenamento. Trate o token como secreto; não compartilhe capturas de tela com o valor visível. Se a API bloquear chamadas da extensão (CORS/regras de origem), use só copiar o link ou um proxy controlado por você.
